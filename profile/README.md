# RustStash

A next-generation gaming platform with Steam integration and cryptocurrency support.

## Features

- Real-time multiplayer games (Jackpot, Coinflip, Crash, Roulette)
- Steam inventory trading and authentication
- Multi-chain cryptocurrency deposits
- Live chat with community features

## Tech Stack

| Layer | Technologies |
|-------|--------------|
| Frontend | Vue 3, Vite, Vuetify |
| Backend | Node.js, Express, Socket.IO |
| Database | MySQL, Redis, PostgreSQL |
| Infrastructure | Docker, Kubernetes |
